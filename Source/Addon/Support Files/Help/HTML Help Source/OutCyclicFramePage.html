Protection and Rolling Counter Outgoing Cyclic Frame

Protection and Rolling Counter Outgoing Cyclic Frames contain signals from the highlighted frame.

What do you want to do?

Set Disable Transmission Channel

The Disable Transmission channel will stop all CAN generation for the given frame when the channel is not set to 0. When the channel is 0, transmission will occur as expected. To set the Disable Transmission channel, select Disable Transmission and then browse to the desired channel.

Delete

To delete a frame, right-click the desired frame in the configuration tree and select Delete from the shortcut menu.

Delete multiple signals

To delete multiple signals, right-click the desired frame in the configuration tree and select Delete Multiple signals from the shortcut menu.
