Protection or Rolling Counter Information Signals

Invalid Count is the number of times a counter or protection signal did not match what was expected.

For a counter signal, Value is the value received on the bus.

For a protection signal, Value is calculated based on the configured protection algorithm and relevant channel values received on the bus.

For example, if the Value received on the bus from a counter signal is not previous Value +1, the Invalid Count will increment. Similarly, if the Value calculated for a protection signal does not match what is received on the bus, the Invalid Count will increment.
